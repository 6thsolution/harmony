# harmony_login

Harmony Login UI Elements

## Installation

add this library to your pubspec.yaml.

```yaml
dependencies:
  harmony_login_ui: latest.version
```

import harmony_log.

```dart
import 'package:harmony_login_ui/harmony_login_ui.dart';
```

## Perquisite

First you need to configure:

- harmony_auth
- harmony_fire
- harmony_login
